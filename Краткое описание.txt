Описание работы бота
1. Регистрация пользователя
При запуске бота, пользователю будет предложено пройти регистрацию, отвечая на несколько вопросов:

Как вас зовут?
Сколько вам лет?
На каком курсе вы учитесь?
Укажите ваш пол (м/ж)
Эти данные будут сохранены в базе данных.

2. Команда /stats
Пользователи могут отправить команду /stats для получения статистики по зарегистрированным студентам:

Количество студентов по возрасту и курсу.
Количество студентов по полу на каждом курсе.
Пример работы с ботом:
После старта бот спросит имя пользователя.
Затем бот спросит возраст, курс и пол, и сохранит эти данные в базе.
Пользователь может отправить команду /stats для получения статистики по студентам.
Структура базы данных
Бот использует SQLite для хранения данных о студентах. База данных содержит таблицу users с полями:

id: Уникальный идентификатор студента (INTEGER, PRIMARY KEY AUTOINCREMENT).
name: Имя студента (TEXT).
age: Возраст студента (INTEGER).
gender: Пол студента (TEXT).
course: Курс студента (TEXT).
